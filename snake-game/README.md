# 🐍 Snake Game em Python

Um clássico jogo da cobrinha desenvolvido com Python e Pygame.  
Simples, divertido e ótimo para aprender programação de jogos! 🎮

---

# 🚀 Como Usar

## 📥 1. Instalação do Python

Primeiro, instale o Python no seu computador através do site oficial:

🌐 :[[python.org]https://](https://www.python.org/)

Após instalar, verifique se está funcionando corretamente.

Abra o **CMD** e execute:

```bash
python --version
```

Se aparecer a versão do Python, está tudo certo ✅

---

# 🎮 2. Instalação do Pygame

No **CMD**, execute o comando abaixo para instalar a biblioteca de jogos do Python:

```bash
pip install pygame
```

---

# 📁 3. Criação do Arquivo

1. Crie uma pasta chamada:

```bash
SNAKE-GAME
```

2. Dentro da pasta, crie o arquivo:

```bash
Snake_game.py
```

3. Copie o código do arquivo `Snake_game.py` deste repositório e cole dentro do seu arquivo criado.

---

# 🖥️ 4. Criação do Atalho

1. Crie um atalho na área de trabalho.
2. Selecione a pasta `SNAKE-GAME`.
3. Adicione o arquivo `Snake_game.py` ao atalho.

---

# ▶️ Executando o Jogo

Abra o arquivo pelo atalho ou execute no CMD:

```bash
python Snake_game.py
```
