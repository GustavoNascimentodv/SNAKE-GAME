Como usar: 

 + ------------------**INSTALAÇÃO DO PYTHON**-----------------
 instale o python no seu computador em **python.org**, verifique 
 se está instalado e funcionando usando *python --version* no CMD.

+ ------------------**INSTALAÇÃO DO PYGAME**------------------
 no CMD execute o comando *pip install pygame* para instalar a biblioteca de jogos do python.

+ ------------------**CRIAÇÃO DO ARQUIVO**--------------------
 Crie uma pasta **SNAKE-GAME**, dentro da pasta crie o arquivo *Snake_game.py*.
 Copie o código do arquivo *Snake_game.py* neste repositório e
 cole dentro do seu aquivo *Snake_game.py*.

+ ------------------**CRIAÇÃO DO ATALHO**---------------------
Crie um atalho na área de trabalho, selecione a pasta **SNAKE-GAME**
adicione o arquivo *Snake_game.py* ao atalho.
